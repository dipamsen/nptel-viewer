# NPTEL Viewer

Alternate client to browse NPTEL Courses and resources.

## Tech Stack
- Backend: Deno Serverless (consuming NPTEL's web API)
- Frontend: React (TypeScript) + React Router + Material UI
- Tooling: Vite, Vercel
